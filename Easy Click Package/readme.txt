Easy Click Installer Setup and Deployment Tool

Set-up Instructions:

1. Before Installing the package, download and install the 7-zip.exe. You may dowanload in here: https://www.7-zip.org/download.html
2. Also check if .Net Framework 4 is installed on your machine. If not, download and install the necessary file. Check
   Microsoft Website: https://www.microsoft.com/en-US/Download/confirmation.aspx?id=17718
3. After installing dependencies (7zip and .Net Framework 4), you can now install the package exe.
4. Enjoy and don't forget to leave suggestions and feedbacks. Thanks!
